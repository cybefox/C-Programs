## Why These Languages?

| Language | Cybersecurity Use Case |
|----------|------------------------|
| **C / C++** | Exploit development, buffer overflows, kernel-level programming, malware analysis |
| **Bash / PowerShell** | Scripting, automation, post-exploitation, CTF tooling |
| **JavaScript** | XSS, browser exploitation, web app pentesting |
| **Python** | Exploit scripting, tool development, fuzzing, automation |
| **SQL** | SQL injection, database exploitation |
| **Assembly** | Shellcode writing, reverse engineering, binary exploitation |
| **Ruby** | Metasploit module development |
| **Golang** | Writing fast, portable security tools and implants |
| **Rust** | Memory-safe systems programming, secure tool development |
| **Solidity** | Smart contract auditing, blockchain security |
